Aplicativo desenvolvido para o controle de consumo de água diário de uma pessoa.

É um app simples que consiste em criações de várias Views em uma tela só para exibir informações para o usuário.

O Botão "beber" chama uma função que altera as states do que está sendo exibido na tela.

Aplicativo utilizado para fins de estudo e livre para qualquer modificação.
